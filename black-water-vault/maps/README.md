# Blackwater Vault Map Holding Area

This folder is the public containment area for final map exports.

Recommended structure:

```text
maps/
  sjrwmd/
    sjrwmd-regional-overview.png
    sjrwmd-regional-detail.png
    sjrwmd-regional-map.pdf
  statewide/
    florida-development-wetlands-overview.png
    florida-development-wetlands-map.pdf
  sfwmd/
    README.md
```

## Launch rule

One polished map is enough to launch the archive.

The first priority is the SJRWMD regional map package:

1. Full-region overview image
2. One readable local/detail image
3. PDF export when ready
4. Short source/method note

## File guidance

Use web-friendly versions here. Keep heavy GIS source files off the public site unless there is a clear reason to publish them.

Good public formats:

- PNG
- JPG
- PDF
- GeoJSON only for lightweight web layers

Avoid uploading huge rasters, raw DEMs, or unfinished processing files here.
